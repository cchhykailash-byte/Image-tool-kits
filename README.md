# 🖼️ Image Toolkit

A complete Image Processing Toolkit built using Streamlit.

## Features

- Image Compressor
- Background Remover
- Image Resizer
- Crop Image
- Rotate Image
- Convert JPG ↔ PNG ↔ WEBP
- Image to PDF
- PDF to Image
- Watermark
- QR Code Generator
- Barcode Generator
- Passport Photo Maker
- Brightness & Contrast
- Blur
- Sharpen
- Color Picker

## Installation

```bash
git clone https://github.com/USERNAME/image-toolkit.git

cd image-toolkit

pip install -r requirements.txt

streamlit run app.py
```

---

## Deploy on Render

### Build Command

```bash
pip install -r requirements.txt
```

### Start Command

```bash
streamlit run app.py --server.port=$PORT --server.address=0.0.0.0
```
